Bienvenue dans l'équipe de développement ! Vous allez réaliser le premier Sprint de notre nouvelle application bancaire web en utilisant **Streamlit** et **Python**, connectés à une base de données **PostgreSQL**.

## 🎯 Votre Mission : Le Sprint 1

Vous devez développer l'application en autonomie en utilisant une **Intelligence Artificielle (IA)** pour vous aider à coder. Votre objectif est de valider les 4 User Stories (US) suivantes :

*   **US1 (Data) :** Je veux une application bancaire qui affiche mes transactions et mon solde.
*   **US2 (Sécurité) :** Je veux pouvoir me connecter et ne voir *que* mes données.
*   **US3 (Fonctionnalité) :** Je veux pouvoir contacter le service client (via un formulaire ou un bouton).
*   **US4 (Design) :** Je veux respecter la charte graphique BNPP (couleurs vert/blanc, logo, style épuré).
